Ця лабораторна робота навчає базовим навичкам взаємодії зі спеціальною мовою стилю сторінок CSS
Виконуючи цю роботу я познайомився з такими засобами для роботи в css:

Селектори:

класів, ідентифікаторів, дітей(дочірній), контекстний(спадкоємців), атрибутів

Псевдокласи

::first-child, :visited, ::lastchild, ::first-of-type, :focus, ::active

Псевдоелементи

::link, ::first-line

а такоже інші параметри для стилізації елементів згідно селекторів


при виконанні роботи я використав усі засоби з перелічених засобів

Висновок:

Виконавши роботу я засвоїв основні навички взаємодії зі спеціальною мовою стилю сторінок css. Зрозумів що таке css та як цого використовувати для зміни стилізації сторінок
